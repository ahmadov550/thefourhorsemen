Learning with LEGO® Mindstorms® EV3 robot


Robotics Project is the last project on Intro to ICT course. Aim of this project to teach and show students how to create a program for a robot. Each group is asked to write a code that make robot to move on the black line, stop whenever there is an obstacle and after removing obstacle continue moving.




First of all, all team members discussed how to write a code, watched tutorials that were given in the instructions. Then we took a robot from Baku European Lyceum for checking whether our code is working or not. For now there are some minor errors in our code. We are working on these problems. When we finish we will publish and share video and reports in all social medias.
